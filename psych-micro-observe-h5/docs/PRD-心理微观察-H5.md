# 产品需求文档（PRD）

## 心理微观察 · H5 测评（525 心理健康节）

| 项目 | 说明 |
|------|------|
| 文档版本 | 1.0（对齐当前前端实现） |
| 产品形态 | 移动端 H5，单页多视图（全屏步骤流） |
| 联名露出 | 测测 × RTC 微颗粒心理实验室 |
| 技术说明 | 本文档描述**当前演示版**交互与规则；收银台、成团、报告下发以**正式后端/支付**为准 |

---

## 1. 背景与目标

### 1.1 产品定位

通过「微颗粒」隐喻与沉浸式动效，引导用户完成**身份选择 → 量表答题 → 直觉猜动物人格 → 阅读报告**，传递自我觉察与心理健康关注（525 心理健康节主题）。

### 1.2 核心目标

- **完成率**：流程清晰、单步单屏，降低跳出。
- **价值感知**：报告分层（自我 / 他人视角 / 建议），猜中动物人格时对「自我分析免费」形成强正向反馈。
- **商业化预留**：按「是否猜中」分支展示不同付费档位与成团优惠文案；前端当前为 **Alert 演示**，对接真实支付时需替换为收银台与订单域逻辑。

### 1.3 非目标（当前演示版）

- 真实现网支付、拼团状态机、用户账号与订单查询。
- 题目与动物算法的临床效度验证（文案宣称「基于量表与算法」为产品包装，正式环境需法务/学术审核）。

## 3. 信息架构与页面流

整体为 **5 个视图（View）**，通过 `showView(viewName)` 切换；**落地页不展示**顶部四步导航，进入「身份」后展示。

| 顺序 | view 名 | 用户可见名称（顶栏） | 说明 |
|------|-----------|----------------------|------|
| 1 | `landing` | （无顶栏） | 品牌与 CTA，星云背景 |
| 2 | `identity` | 身份 | 选择测评所锚定的身份 |
| 3 | `quiz` | 答题 | 70 题 Likert，进度环 |
| 4 | `guess` | 猜颗粒 | 6 选 1 动物人格，强调惊喜与猜中免费自我分析（**不展开具体付费细则**） |
| 5 | `report` | 报告 | 三大块报告 + 底部固定付费坞 |

**导航规则**

- 身份页：可返回落地页。
- 答题页：可返回身份页（重选身份会重新开始答题的状态由当前实现：`btnStartQuiz` 会重置答题进度；从答题返回身份不自动清状态，需产品确认是否需「退出测评确认」）。
- 猜颗粒、报告：按当前实现无单独「返回上一步」按钮（若产品需要需补需求）。

## 4. 功能需求

### 4.1 落地页（Landing）

**入口**

- 主按钮：「开启灵魂高倍扫描」→ 进入身份页；可配合音效/环境音（`MicroAudio`，以实际实现为准）。

**内容要素**

- 525 心理健康节、联名标识（测测 × RTC）。
- 主标题含年份（当前文案为 2026）、副标题。
- 信任背书：量表与算法说明、用户数量展示（演示数据）。
- 隐私说明：结果加密、仅本人可见。

**动效**

- 星云 Canvas（非答题页时运行，进入其他页停止以省资源）。

### 4.2 身份定频（Identity）

**必选**：用户须选择一种身份后方可「开始 70 题采样」。

**身份列表（当前实现）**

| id | 展示名 | 说明 |
|----|--------|------|
| self | 自己 | 核心颗粒 |
| wife / husband | 妻子 / 丈夫 | 情感颗粒；选中时展示双人剪影装饰 |
| mother / father | 母亲 / 父亲 | 防御颗粒 |
| staff / manager | 基层员工 / 管理者 | 能量颗粒 |
| student | 学生 | 成长颗粒 |

每条身份含：扫描主题句、一句钩子文案；选中后高亮并写入全局 `--identity-accent` 色。

**主按钮**

- 文案：`开始 70 题采样`；未选身份时禁用。

### 4.3 答题（Quiz）

**题量与题型**

- 共 **70** 题（`MICRO_QUESTIONS.length`），单选 **1～5 分** Likert（非常不同意 → 非常同意）。
- 部分题目支持反向计分（`inv`），由题库定义，前端在 `recordAnswer` 中做 `6 - score` 处理。

**维度**

五维标签（用于雷达与计分）：

- 自我觉察、人际亲和、情绪稳定、规则计划、开放探索  

每题归属某一维度 `dim`，答题累加该维分数与题数，全部答完后归一化为 0～100 的 `profile[5]`。

**进度与反馈**

- 顶部展示当前题号 / 总题数、环形进度百分比。
- 答题过程中显微镜模糊度随进度变化（`--lens-blur`）。
- 在第 **20、40、60** 题完成后（即进入下一题索引为 20/40/60 时触发）展示 Toast 鼓励语，约 3.2s 消失。

**操作**

- 点击某一 Likert 选项即**提交本题并进入下一题**（无单独「下一题」按钮）。

### 4.4 猜颗粒（Guess）

**业务叙事（产品要求）**

- 强调 **惊喜感**：揭晓前最后一问；若用户所选与系统判定一致，**「自我性格分析」全文免费展开**。
- **不在本页**展开具体付费档位、成团规则（该部分归属报告页与后端策略）。

**算法与选项**

- 根据五维 profile，用 `pickTrueAnimal` 在 **16 个动物人格**中与用户向量余弦相似度最高者为「真实结果」。
- 选项池：**1 个真实动物 + 从其余 15 个中随机 5 个**，共 **6** 张卡片，顺序洗牌展示。
- 每张卡片：emoji、名称（如「温和的鹿」）、一句话描述。

**交互**

- 用户点选一张卡片为选中态；底部「确认选择」提交。
- 若猜中：弹出「认知天才」全屏层，展示匹配度 100%，约 2.2s 后自动进入报告；用户也可点「查看报告」立即进入。
- 若未猜中：直接进入报告。匹配度为所选动物向量与用户 profile 的余弦相似度（0～100%）。

### 4.5 报告（Report）

报告分 **三部分**：

1. **自我性格分析**（含动物人格结果、雷达、心理健康指数、特质、解析、压力关注点等模块）。
2. **他人眼中的你**（基于所选身份视角的四模块叙事）— 默认 **磨砂遮罩 + 解锁入口**。
3. **后续建议** — 文案生成后仍带锁定区与解锁按钮。

**猜中 / 未猜中 分支（当前前端逻辑）**

| 状态 | 第一部分「自我性格分析」 | 主付费按钮（底部坞） | 成团提示 |
|------|--------------------------|----------------------|----------|
| **猜中** | 全文可见，不显示自我区锁层 | `¥12.9` 解锁「他人眼中的你 + 后续建议」 | 按 **¥12.9** 计算 7 折后人均价 |
| **未猜中** | 整体加锁层（`is-locked`），遮罩提示付 **¥19.9** 解锁完整报告 | `¥19.9` 解锁完整报告 | 按 **¥19.9** 计算 7 折后人均价 |

**价格常量（前端演示）**

- `PRICE_FULL = 19.9`（完整报告）
- `PRICE_OTHERS = 12.9`（仅他人视角 + 后续建议，在猜中后使用）
- `DISCOUNT = 0.7`（成团折扣系数，用于展示「人均约 ¥xx」）

**底部付费坞**

- 说明文案 `reportPayTip`、主按钮 `btnPayPrimary`、次按钮「邀请好友成团 · **立享 7 折**」`btnPayGroupInline`、成团说明 `reportPayGroupHint`。
- 他人模块内 CTA `btnUnlockOther`、建议区 `btnUnlockAdvice` 与主按钮在演示中均走同一套 **Alert**，正式环境应对接统一收银台并区分 SKU。

**分享**

- 「生成分享卡片」：支持 Web Share API，否则降级为复制提示文案。

## 5. 关键状态与数据

| 状态字段 | 含义 |
|----------|------|
| `identity` | 当前身份 id |
| `qIndex` / `answers` / `scores` / `counts` | 答题进度与计分 |
| `profile` | 五维 0～100，答完后计算 |
| `trueAnimal` | 算法选中的动物人格 |
| `selectedAnimalId` | 用户猜测 |
| `guessExact` | 是否猜中 |
| `matchPercent` | 展示用匹配度（猜中固定 100） |
| `reportBundle` | 报告渲染用文案结构体 |

---

## 6. 非功能需求（摘要）

- **布局**：单屏视图 `position: fixed`，主区域可滚动，报告底部坞固定，需兼容 **safe-area**。
- **无障碍**：猜颗粒列表 `listbox` / `option`、`aria-selected`（可按产品标准继续加强）。
- **性能**：非落地页停止星云动画；雷达图为 Canvas 绘制。

---

## 7. 与后端/支付的对接清单（供研发）

1. 提交答题结果 → 服务端存证并计算/确认 `profile` 与 `trueAnimal`（若防篡改需服务端重算）。
2. 提交用户猜测 → 返回是否猜中及匹配度；驱动报告解锁策略。
3. 订单：完整报告 SKU、他人+建议 SKU、成团订单与价格校验（7 折规则以后台配置为准）。
4. 报告正文：演示版为前端模板生成；正式环境可由服务端下发或混合渲染。

---

## 8. 文案与品牌备注

- 猜颗粒页：**不**展示具体金额与成团细则；报告页与底部坞承担付费说明。
- 顶栏四步：**身份 / 答题 / 猜颗粒 / 报告**。
- 报告 kicker 当前为 `Observatory · RTC Lab`（可按品牌规范替换）。
